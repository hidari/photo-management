# 初期設定ガイド

## 概要

写真配布ツールを使用する前に、初回のみ実行する初期設定コマンドです。

実行すると対話的に初期設定を行います。

## コマンド

```bash
deno task setup
```

## 実行内容

このコマンドは以下の3つの初期設定を行います：

### 1. config.tsの作成

`config.example.ts` をテンプレートとして、個人設定ファイル `config.ts` を作成します。

- 管理者情報
- 連絡先情報（SNS、メールアドレスなど）

**注意**: `config.ts` は `.gitignore` に含まれるため Git にコミットされません。公開しないようにしてください。

### 2. アーカイブ作成ツール rip のダウンロード

写真ファイルのZIPアーカイブ化に使用するツール `rip` をダウンロードします。

なお、ZIPアーカイブに使用するツールはあとで `config.ts` を編集することで変更できるためスキップ可能です。

### 3. Google Drive OAuth認証

Google Drive APIを使用するための認証を実行します。

1. ブラウザが自動で開きます
2. Googleアカウントでログイン
3. アクセス許可を承認
4. 認証トークンが自動保存されます

## トラブルシューティング

### Google Drive認証に失敗する場合

1. `config.ts` の Google Drive API 認証情報（Client ID、Client Secret）が正しいか確認
2. Google Cloud Consoleで該当プロジェクトのAPIが有効化されているか確認
3. `~/.config/photo-management/google-drive-token.json` を削除して再認証を試みる

## 次のステップ

初期設定が完了したら、次は[イベント作成ガイド](./イベント作成ガイド.md)を参照して新しいイベントを作成してください。
